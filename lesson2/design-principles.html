Visual Hierarchy

w3schools.com

The w3schools website is a great example of Visual Hierarchy. On the home page the goal of the website is very clear because the words "Learn To Code" are the largest. The next biggest elements of the page are the search engine and the words "Not Sure Where To Begin?" giving great access to the rest of the website.

Rule of Thirds

LDS.org

Right now the home page of the Church of Jesus Christ of Latter-day Saints website has a great photo example of the Rule of Thirds. As you can see in the picture, the man Jesus is interacting with is lined up with the left perpendicular line and both their heads line up with the top horizontal line.

PARC: Contrast

GitHub.com

GitHub's website uses a lot of visual contrast, in the picture we can see that they use a dark-colored background to help make the content pop, using white, light gray, and light purple as colorful aids to help make the contents of the page easy to read and see.
